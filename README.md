# Hatena

This is prototype version of Ruby inteface for Hatena API.

WARNING: This gem is the rough version for private use.

## Installation

Add this line to your application's Gemfile:

    gem 'hatena'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hatena
